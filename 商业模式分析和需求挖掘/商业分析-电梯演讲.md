# 4. **为什么需要电梯演讲**

- 电梯演讲提供一个框架帮助你组织想法，便于你表达,例如：当你想要向老板呈现你对产品的想法时.

- 你的好创意不光激励你，而且要对其他人也激动人心,例如：当你想要你的团队充分理解你想要做的产品功能时.

- 电梯演讲帮助你始终关注实际的用户和市场需要，不沉迷于不切实际的想法和无休止的功能.

# 5. **电梯演讲对产品经理的价值**
```电梯演讲是产品的北极星，产品经理参考电梯演讲对功能做优先级和排期```
